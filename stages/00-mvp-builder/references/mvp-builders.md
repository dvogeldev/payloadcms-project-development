# MVP Builders Reference

Online builders and LLM tools for rapid prototyping:

## Recommended Tools
- lovable.dev - AI-powered full-stack app builder
- v0.dev - Generative UI component builder by Vercel
- bolt.new - AI web development agent
- antigravity - Rapid prototyping platform
- Plain LLM prompts - Using Claude, GPT-4, etc. to generate code

## Usage Guidelines
1. Start with a clear project description
2. Generate initial MVP
3. Test core user flows
4. Gather feedback on usability and features
5. Refine requirements based on MVP learnings
